# Outline PPT: Git dan GitLab (2 Jam)

## Panduan Waktu
- Total: 120 menit.
- Slide 1-3: 10 menit.
- Slide 4-16: 55 menit (termasuk 2 demo).
- Slide 17-27: 40 menit.
- Slide 28-30: 15 menit.

## Slide-by-Slide

1. Sampul
Durasi: 1 menit
Tujuan: Membuka sesi dan konteks pelatihan.
Konten: Judul, nama pengajar, tanggal.

2. Tujuan Pembelajaran
Durasi: 2 menit
Tujuan: Menyelaraskan ekspektasi.
Konten: 5-6 tujuan utama sesi.

3. Agenda 2 Jam
Durasi: 2 menit
Tujuan: Memberi peta waktu.
Konten: Alur topik dan demo.

4. Masalah Tanpa Version Control
Durasi: 3 menit
Tujuan: Membangun urgensi.
Konten: Risiko folder versi manual, hilang histori.

5. Apa Itu Version Control
Durasi: 3 menit
Tujuan: Definisi singkat dan manfaat.
Konten: Histori, kolaborasi, rollback.

6. Apa Itu Git dan Alternatifnya
Durasi: 3 menit
Tujuan: Memahami Git dalam ekosistem.
Konten: Git (DVCS), SVN, Mercurial, Perforce.

7. Git sebagai Dokumentasi
Durasi: 3 menit
Tujuan: Menjelaskan nilai commit.
Konten: Commit sebagai snapshot dan cerita perubahan.

8. Konsep Dasar Git
Durasi: 4 menit
Tujuan: Memahami istilah inti.
Konten: Repo, commit, branch, merge, remote.

9. Alur Kerja Git Sederhana
Durasi: 4 menit
Tujuan: Menunjukkan flow kerja.
Konten: Branch -> commit -> push -> PR -> merge.

10. Setup Cepat
Durasi: 3 menit
Tujuan: Menyiapkan dasar.
Konten: `git config`, cek versi.

11. Perintah Inti Git (Bagian 1)
Durasi: 4 menit
Tujuan: Mengenal perintah dasar.
Konten: `git init`, `git status`, `git add`, `git commit`.

12. Demo 1: Repo Lokal Cepat
Durasi: 8 menit
Tujuan: Praktik langsung.
Konten: Init repo, buat file, commit, lihat log.

13. Perintah Inti Git (Bagian 2)
Durasi: 4 menit
Tujuan: Melengkapi kebutuhan harian.
Konten: `git log`, `git diff`, `git branch`, `git switch`, `git checkout`.

14. Konvensi Commit
Durasi: 3 menit
Tujuan: Komunikasi perubahan yang rapi.
Konten: Commit kecil, deskriptif, imperatif.

15. Branching dan Merge
Durasi: 4 menit
Tujuan: Konsep kerja paralel.
Konten: Branch untuk fitur, merge untuk integrasi.

16. Demo 2: Konflik Singkat
Durasi: 8 menit
Tujuan: Menunjukkan penyelesaian konflik.
Konten: Edit baris sama, merge, resolve.

17. Terminal vs GUI
Durasi: 3 menit
Tujuan: Menunjukkan opsi tools.
Konten: CLI vs GitHub Desktop/VS Code.

18. Git vs GitLab
Durasi: 3 menit
Tujuan: Memisahkan konsep.
Konten: Git = alat, GitLab = platform.

19. Fitur Dasar GitLab
Durasi: 4 menit
Tujuan: Pengenalan ekosistem.
Konten: Repo, issue, MR, review, boards.

20. Local vs Remote
Durasi: 4 menit
Tujuan: Memahami arah data.
Konten: Push, pull, origin.

21. GitLab Issues
Durasi: 3 menit
Tujuan: Manajemen pekerjaan.
Konten: Issue sebagai tiket kerja.

22. Merge Request
Durasi: 4 menit
Tujuan: Review dan kolaborasi.
Konten: MR sebagai proposal perubahan.

23. Forking
Durasi: 2 menit
Tujuan: Kontribusi open source.
Konten: Fork untuk repo yang bukan milik kita.

24. Panduan Branch-Issue-MR
Durasi: 4 menit
Tujuan: Alur kerja ringkas.

25. Merge, Squash, Rebase

26. GitLab CI/CD (Intro)

Tujuan: Otomatisasi dasar.

27. Markdown Basics
Durasi: 3 menit
Tujuan: Dokumentasi ringkas.
